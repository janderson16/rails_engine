# README

This application creates an API for a series of merchants and their transactions, invoices, items, and customers.

This application uses:
Ruby version - 2.3.0
Rails version - 5.0.2
PostgreSQL

In order to run:
-Clone repository
-Run 'bundle exec' from command line
-Run 'rake db:create'
-Run 'rake db:migrate'
-Run 'rake db:seed'
